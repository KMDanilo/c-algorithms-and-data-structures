# C Programming: Algorithms & Data Structures

This repository contains a comprehensive collection of C programs, mathematical algorithms, and data structures. I developed these exercises and projects throughout my Computer Science degree to build a strong foundation in memory management, pointers, and algorithmic efficiency.

> **Note:** Because these exercises were originally developed for local university coursework, some source code variables, comments, and terminal outputs or attached PDFs are in Portuguese. This README was translated into English to better showcase the learning progression, but the core logic, math, and data structures functions!

## Repository Structure

The project is logically divided into four main areas, scaling from basic programming logic to more complex, non-linear data structures:

### 1. Fundamentals of C (`01-Fundamentals-of-C`)

This section covers the basics of the C programming language and structural logic.

* **Topics covered:** Basic I/O operations, Conditionals, Loops, Functions, Arrays, Matrices, Strings manipulation.
* **Advanced basics:** Direct memory access with Pointers, Dynamic Memory Allocation and File Handling.

### 2. Linear Algebra (`02-Linear-Algebra`)

Mathematical applications and numerical algorithms built in C.

### 3. Data Structures I (`03-Data-Structures-I`)

Implementations of fundamental Abstract Data Types (ADTs) and linear memory management.

* **Structures:**
  * Singly Linked Lists
  * Stacks (LIFO)
  * Double-linked Queues (FIFO)
* **Sorting Algorithms:** Classic sorting implementations.

### 4. Data Structures II (`04-Data-Structures-II`)

Advanced, non-linear data structures focused on high-efficiency searching, sorting, and hierarchical mapping.

* **Trees:** AVL Tree.
* **Graphs:** Graph representations.
* **Heaps:** Min/Max Heaps for priority queues.
* **Hash Tables:** Implementations using both Open Addressing and Separate Chaining.
